# 제로트래픽 통합판 — 대형시설 진입로·주차 혼잡도

코스트코·트레이더스 개별 사이트를 하나로 통합한 버전. 첫 화면에서 카테고리 블록을 고르면
지점별 진입로 혼잡도(네이버)와, 공항은 주차장 층·구역별 잔여 대수(공공데이터)를 보여준다.

## 카테고리 (첫 화면 블록 12개)

코스트코(20) · 트레이더스(24) · 아울렛(9) · 스타필드(6) · 이케아(4) · 백화점·복합몰(6: 더현대서울·롯데월드몰·신세계강남·현대판교·롯데몰김포공항·센텀시티) · 에버랜드·롯데월드(2) · 워터파크(3: 캐리비안베이·오션월드·롯데워터파크) · 전시장(3: 킨텍스·코엑스·벡스코) · 야구장(3: 잠실·고척돔·랜더스필드) · 대형병원(5: 아산·삼성서울·세브란스·성모·분당서울대) · 공항 주차장(5: 인천 T1/T2·김포·제주·김해)

특성이 다른 카테고리는 화면 상단에 안내 배너가 뜬다(야구장=경기일만 혼잡, 병원=평일 오전 피크, 전시장=행사일, 워터파크=여름 성수기). 인천공항 카드에는 주차장 잔여 대수 아래에 **이 시간대 예상 출국객 수**(승객예고 API)도 함께 표시된다.

## 구조

```
zerotraffic-app/
├─ public/
│  ├─ index.html          # SPA (홈 타일 → 카테고리 카드 → 탭하여 조회)
│  └─ og.png              # 링크 미리보기 (1200x630)
├─ netlify/functions/
│  ├─ store.mjs           # [HTTP] ?cat=&name= 진입로 혼잡도 (네이버 온디맨드)
│  ├─ airport.mjs         # [HTTP] ?code= 공항 주차장 잔여 대수 (공공데이터)
│  └─ _shared.mjs         # 카테고리·장소 데이터, 헬퍼
├─ netlify.toml
└─ package.json           # 의존성 없음
```

## 동작 원칙 (코스트코판과 동일)

- 온디맨드: 카드를 탭할 때만 외부 API 호출. 마트류 10분 캐시, 공항 5분 캐시.
- 카테고리별 영업시간 외 차단(아울렛 10~21시, 테마파크 9~22시, 그 외 10~22시, 공항 24시간).
- IP당 1분 40회 레이트리밋, 목록에 없는 장소는 404 → 비용 상한 고정.
- 키 없거나 API 실패 시 "임시값" 표시로 데모 폴백(화면이 죽지 않음).

## 환경변수 (Netlify → Environment variables)

| 변수 | 용도 |
|---|---|
| `NCP_CLIENT_ID` / `NCP_CLIENT_SECRET` | 네이버클라우드 Maps (기존 키 재사용) |
| `DATA_GO_KR_KEY` | 공공데이터포털 인증키 (공항 주차장용, 아래 참고) |

### 공항 API 신청 방법 (무료)

1. data.go.kr 가입 → 아래 3개 API "활용신청" (자동승인, 무료):
   - 인천국제공항공사_주차 정보 (data.go.kr/data/15095047)
   - 한국공항공사_전국공항 실시간 주차정보 (data.go.kr/data/15056803)
   - 인천국제공항공사_승객예고-출·입국장별 (data.go.kr/data/15095066) ← 출국장 혼잡 표시용
2. 마이페이지에서 "일반 인증키(Decoding)"를 복사 → Netlify 환경변수 `DATA_GO_KR_KEY`에 등록.
3. 키 하나로 두 API 모두 사용된다. 확인: `사이트주소/.netlify/functions/airport?code=GMP`
4. ※ 한국공항공사 API는 기관 사정으로 응답 필드명이 바뀌는 경우가 있다. 임시값이 계속 뜨면
   `airport.mjs`의 `pick()` 후보 필드명에 실제 응답 필드를 추가하면 된다.

## 배포

1. 이 폴더를 Netlify Drop으로 새 사이트에 올리거나, **기존 zerotraffic 사이트에 덮어쓰기 배포**(권장 — 도메인 유지, 기존 홍보 링크 그대로 살아있음).
2. 환경변수 3개 등록 → Trigger deploy.
3. 네이버 Application의 Web 서비스 URL에 주소 등록(기존 zerotraffic이면 이미 되어 있음).

## 토스 앱인토스(미니앱) 등록

- 이 앱은 모바일 반응형 웹 + PWA(manifest·아이콘·safe-area 대응)라 웹뷰 미니앱에 그대로 얹을 수 있다.
- **사업자 등록 없이도 미니앱 출시 가능**(2026 기준, 앱인토스 공지). 단 토스 로그인·인앱 광고·
  토스페이·프로모션 등 수익화 기능은 사업자 등록 후 사용 가능. 콘솔 가입 조건: 만 19세 이상 + 본인 명의 토스앱.
- 등록: developers-apps-in-toss.toss.im → 콘솔에서 앱 등록 → 이 사이트 URL 연결.
- 수익화(1단계 인앱 광고)를 붙이려면 그때 사업자 등록이 필요하다. 개인사업자는 업종이
  미니앱 서비스 업종과 일치해야 한다.
